r8er — 1080p + subtitles through the tunnel

Jellyfin remuxes the 1080p clip to HLS and delivers the subtitle as a WebVTT rendition; hls.js plays it, but every playlist, segment, and subtitle is fetched over the WebRTC data channel via a custom loader — no direct path to Jellyfin. Run setup-jellyfin.sh on the home machine and open the phone URL it prints (#item=…&key=…&subs=…). Relay-needing networks: append the same &turn=&user=&cred= you use on connect.html.
