r8er — real HLS playback through the tunnel

Jellyfin transcodes the clip to HLS; hls.js plays it, but every playlist and segment is fetched over the WebRTC data channel via a custom loader — no direct network path to Jellyfin. Start the agent, run setup-jellyfin.sh, then fill in the item id + api key it printed (or open this page with #item=…&key=… in the URL).
